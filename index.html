Typogruby

A simple Ruby gem to improve your web typography.

  • Home

Typogruby can apply a set of text filters to your HTML documents to make your text look better. It is based on Django’s typogrify.

Usage

Typogruby is a Ruby gem and as such requires Ruby installed on your system. Mac OS X and *nix systems usually come with Ruby installed by default.

To install the gem, open a terminal window and run:

gem install typogruby

You can now use typogruby in two ways:

  1. Apply it to a file in the terminal:

    typogruby index.html

    Type typogruby -h for more information.

  2. Use it from a Ruby script:

    {% highlight ruby %} require 'typogruby' puts Typogruby.improve('Hello, world!') {% endhighlight %}

Included filters

  1. Rubypants is used for nice special characters, such as quotes and dashes

  2. A port of Shaun Inman’s widon’t is used to prevent widows (using a &nbsp; to prevent a single word on a new line).

  3. All consecutive capital letters are wrapped in span.caps

  4. All quotes at the start of a line are wrapped in span.dquo

  5. All ampersands (&) are wrapped in span.amp
